Há duas formas possíveis de inserir um áudio em sua página com o elemento audio.

A primeira forma é inserindo um áudio utilizando o atributo src (origem) diretamente no elemento audio conforme o exemplo abaixo.

A segunda forma é inserir um áudio utilizando o elemento source (origem) entre as tags de abertura e fechamento do elemento audio. Utilizamos esta forma sempre que quisermos subir mais de umz extensão do mesmo áudio, para garantir que ele será corretamente carregado em mais de um navegador ou mesmo para que o navegador tenha a opção de escolher qual áudio ele reconhece.
